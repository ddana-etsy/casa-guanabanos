## 14. Recomendación

Para una casa de alta gama en Cuajimalpa con enfoque en eficiencia de costo sin perder calidad:

**Estrategia recomendada: Opción A como base + Opción B en puntos de impacto**

| Categoría | Opción Recomendada | Razón |
|-----------|--------------------|-------|
| Pisos sociales | **B** (mármol natural) | Es lo que tus invitados ven y tocan. Define el nivel de la casa. |
| Pisos recámaras | **A** (ingeniería de roble) | Kährs o Haro se ven y sienten igual que madera maciza a mitad de precio. Compatible con piso radiante. |
| Pisos sótano | **A** (concreto pulido) | Nadie espera mármol en el estacionamiento. |
| Pisos terraza | **A** (porcelánico exterior) | Mejor mantenimiento que madera, resiste clima CDMX. |
| Escaleras | **B** | Elemento arquitectónico central — invertir aquí. |
| Pintura | **A** (Comex Premium) | Benjamin Moore no justifica 3× el costo para la mayoría de los muros. |
| Muros acento | **B** (lambrin madera) | Alto impacto visual en puntos focales. |
| Cocina gabinetes | **A o B** | Aran Cucine tiene calidad italiana real. Poggenpohl si el presupuesto lo permite. |
| Electrodomésticos | **B** (Miele) | Miele es el estándar de la industria. Mejor servicio técnico en MX que Sub-Zero. |
| Baño principal | **B** | Hansgrohe Axor + Villeroy & Boch = lujo legítimo sin el sobreprecio de Dornbracht. |
| Baños secundarios | **A** | Duravit + Hansgrohe es excelente calidad para baños que no son el showcase. |
| Puertas | **A con puerta principal B** | Interiores carpintería local + puerta de acceso como statement piece. |
| Ventanería | **B** (Schüco) | Inversión que define eficiencia térmica, acústica Y estética. No escatimar. |
| Luminarias | **A empotradas + B decorativas** | WAC/Lucifer para lo funcional, Flos/Artemide para las piezas de declaración. |
| Fachada | **B** | La primera impresión. Piedra natural + concreto aparente = atemporal. |
| Closets | **A** | Carpintería local bien ejecutada con herrajes Hafele iguala a Poliform visualmente. |

**~$5,500,000 – $7,500,000 MXN ($314,000 – $429,000 USD)**
